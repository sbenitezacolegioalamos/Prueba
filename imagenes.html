SVG

SVG o Gráficos Vectoriales Escalables es un estándar web para definir gráficos basados en vectores en páginas web. Basado en XML, el estándar SVG proporciona marcado para describir rutas, formas y texto dentro de una ventana gráfica.

Webp

Está pensado especialmente para su uso en Web, ya que reduce el tamaño de las imágenes y, por consiguiente, el tiempo de carga, mejorando la velocidad y la experiencia de navegación.

GIF

Un GIF es un formato de imagen, que soporta como máximo 256 colores y reproduce de forma consecutiva una serie de imágenes en segundos, comprendidos entre 5 y 10.

PNG

El formato PNG permite generar imágenes de mapas de bits con una elevada 'ratio' de compresión de tamaño.

JPG

El archivo JPG reduce el tamaño de la foto mediante un sistema de compresión que elimina la información innecesaria mediante un procedimiento de comparación de píxeles cercanos.
