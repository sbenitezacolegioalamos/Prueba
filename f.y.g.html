Flexbox y Grid
¿Qué es flexbox? Es un modelo unidimensional de layout, sirve para distribuir el espacio entre los ítems de una interfaz y mejorar la alineación. Tiene como meta hacer una forma más eficiente para crear diseños.
¿Qué es grid? Es una técnica o sistema de maquetación web de las hojas de estilo en cascada que divide la página en una cuadricula o rejilla, permite diseños complejos y adaptables.
